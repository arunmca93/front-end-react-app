## PrimeHomes Realty - Simple React App

This is a simple, modern React single-page app for a real estate site, built with Vite, TypeScript, and Tailwind CSS.

### Scripts

- **Install dependencies**

```bash
npm install
```

- **Run in development**

```bash
npm run dev
```

- **Build for production**

```bash
npm run build
```

- **Preview production build**

```bash
npm run preview
```

### Features

- **Hero section** with branding and key stats.
- **Search bar** with filters for location, property type, and price range.
- **Featured property cards** with price, beds/baths, area, and badges (Featured, New, For Rent).
- **Responsive layout** that looks good on mobile and desktop.
